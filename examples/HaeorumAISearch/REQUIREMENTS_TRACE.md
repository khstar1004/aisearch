## 현재 로컬 인수 상태

아래 명령이 통과하면 로컬 샘플 기준 1차 기능은 회귀 없이 동작하는 것으로 봅니다.

```powershell
python examples\HaeorumAISearch\scripts\contract_check.py
python -m unittest discover examples\HaeorumAISearch\tests
python examples\HaeorumAISearch\scripts\acceptance_check.py
python examples\HaeorumAISearch\scripts\local_acceptance.py --output examples\HaeorumAISearch\logs\local-acceptance.json --markdown-output examples\HaeorumAISearch\logs\local-acceptance.md
python examples\HaeorumAISearch\scripts\requirements_audit.py --output examples\HaeorumAISearch\logs\requirements-audit.json --markdown-output examples\HaeorumAISearch\logs\requirements-audit.md
```

운영 완료 판단은 위 로컬 검사만으로 하지 않습니다. 실제 Marqo, MSSQL, 이미지 서버, 대표 가맹점 사이트에서 운영 환경 확인 항목을 통과해야 합니다.
