# 릴리스 기록

이 저장소는 기존 Git/GitHub 히스토리를 제거하고 한국어 문서 기준으로 새로 시작하도록 정리되었습니다.

이전 업스트림 프로젝트의 오래된 릴리스 노트는 새 저장소의 변경 이력과 혼동되지 않도록 보관하지 않습니다. 앞으로의 변경 사항은 새 초기 커밋 이후의 커밋, 태그, 릴리스 노트에서 관리하세요.

## 새 릴리스 작성 기준

- 사용자에게 의미 있는 기능 변경, 버그 수정, 호환성 변경을 한국어로 기록합니다.
- API, 환경 변수, Docker 실행 방식이 바뀌면 마이그레이션 메모를 함께 남깁니다.
- 자동 생성된 의존성 갱신만 있는 경우에는 핵심 영향만 짧게 요약합니다.
